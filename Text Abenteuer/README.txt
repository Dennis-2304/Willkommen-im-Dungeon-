
Ein kleines Text-Adventure in Java: Du erkundest einen Dungeon, triffst Entscheidungen, kämpfst gegen Gegner und erlebst unterschiedliche Enden.

Funktionen:

Spielername eingeben

Wege wählen und Aktionen treffen

Einfaches Kampfsystem

Mehrere mögliche Enden

Start:

ZIP über den grünen „Code“-Button herunterladen

Entpacken und Main.java mit Java starten

Oder den Code direkt auf GitHub anschauen, indem du die Ordner öffnest

Viel Spaß beim Ausprobieren!
